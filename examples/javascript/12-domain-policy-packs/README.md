# 12 - Domain Policy Pack Catalog (JavaScript)

Loads signed domain Policy Packs from the catalog, verifies signatures, applies
an override, and runs the resulting scanners through `piiGuardFromPolicyPack`.
**No API key, no network**.

```bash
npm install
npm start
```
