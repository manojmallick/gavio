# Production Agent Framework

Offline production-style ecosystem trust app for v2.7.0. It models a governed
agent workflow that hands metadata-only Gavio evidence to Portkey, Helicone,
LangChain, LangGraph, Vercel AI SDK, and the OpenAI SDK boundary without
importing any external ecosystem SDK.

```bash
PYTHONPATH=packages/gavio-py python examples/integrations/production-agent-framework/app.py
```

No provider key, gateway account, framework runtime, or network access is
required.
